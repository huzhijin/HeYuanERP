# P8 发票（开票/电票/打印）阶段说明

本阶段基于 Plan A（.NET 8 + Vue 3 + SQL Server）实现“从订单/交货生成发票、发票查询与打印（调用 P11）”。

## 内容摘要
- 后端：领域模型（Invoice/InvoiceItem/EInvoiceInfo/状态/来源）、应用服务（创建/查询/状态流转）、打印服务与控制器、仓储、EF 配置与迁移占位、依赖注入配置。
- 前端：发票服务封装、Pinia 仓库、列表页、开票表单、详情页、状态标签、打印按钮、路由。
- 配置：`.env.example` 增加打印服务相关环境变量（PRINTING_*）。
